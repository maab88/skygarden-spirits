# Skygarden Spirits

A beginner-friendly 2D mobile puzzle game built in Godot 4.

## MVP Scope
- Main menu
- Level select
- Gameplay
- Results
- 10 levels
- 4 weather powers: Rain, Sun, Frost, Wind
- Local save progression

## Tech
- Godot 4
- GDScript
- Android first

Project setup in progress.
